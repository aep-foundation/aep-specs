# Code of Conduct

The AEP Foundation community is committed to a professional, respectful, and
collaborative standards-development environment.

## Expected Behavior

- Use clear, constructive technical feedback.
- Assume good faith while holding protocol arguments to a high standard.
- Focus criticism on designs, text, tests, and interoperability impact.
- Respect differences in background, affiliation, and implementation context.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or conduct.
- Sustained disruption of technical discussion.
- Publishing private information without permission.
- Bad-faith participation intended to derail standards work.

## Enforcement

Maintainers may moderate issues, discussions, pull requests, and repository
access to preserve a productive technical environment.

Report conduct concerns privately to:

```text
conduct@aep.foundation
```

Alternatively, contact a repository owner directly through GitHub.
