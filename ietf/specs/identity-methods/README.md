# Identity Method Specifications

Identity method documents define optional Agent identity mechanisms that
Services can enable for authenticated AEP commands.

The core protocol defines the `identity.methods` advertisement and client
assertion verification substrate. Identity method documents define concrete
identifier syntax, verification-material resolution, `kid` handling, caching,
rotation, and trust-anchor behavior.
